# Python codes to compute and visualize point count, RMSE, and ground motion velocity along linear infrastructure.

The codes should be run in QGIS IDE. The name of the layers should be set first. All the layers have been unified with same geographic extent and coordinate systems. The layers could be inported from csv, or shp, or pckg files with ground motion data such as those from the European Ground Motion Service (EGMS) or with export commands in MintPy.

The output of the codes are figures. If you want to save the data, you need to add a command for that.
